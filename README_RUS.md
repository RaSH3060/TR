# Руководство по использованию MKXL Trainer

## Описание программы
MKXL Trainer - это внешнее приложение для игры Mortal Kombat XL, которое позволяет управлять вводом с клавиатуры и других устройств ввода для выполнения сложных комбо и техник. Программа работает путем внедрения DLL в процесс игры и считывания данных из памяти игры.

## Установка и запуск

1. Запустите MKXL Trainer.exe
2. Убедитесь, что игра MKXL (MK10.exe) не запущена
3. Нажмите кнопку "Browse MK10.exe" и выберите исполняемый файл игры
4. Нажмите "Deploy DLL" чтобы скопировать dinput8.dll в директорию игры
5. Запустите игру

## Основные функции

### 1. Конфигурация памяти
- **Базовый адрес**: автоматически установлен на 033C8A98
- **Оффсеты**: 8, 130, 108, 78, 90, 120, F20

### 2. Настройки тайминга
- **Порог блокировки (ms)**: по умолчанию 260ms - при достижении этого значения начнется блокировка определенных клавиш
- **Порог макроса (ms)**: по умолчанию 280ms - при достижении этого значения начнется выполнение макроса

### 3. Блокировка клавиш
- Включите чекбокс "Enable Input Blocking" чтобы активировать блокировку
- Нажмите "Add Blocked Key" чтобы добавить клавишу, которая будет блокироваться во время определенных таймингов
- Блокировка применяется только когда таймер достигает порога блокировки

### 4. Редактор макросов
- Добавьте шаги макроса с помощью кнопки "Add Step"
- Каждый шаг содержит:
  - **Кнопка**: код клавиши для нажатия
  - **Продолжительность (мс)**: как долго удерживать клавишу
  - **Задержка после (мс)**: пауза после отпускания клавиши
- Нажмите "Run Macro" чтобы запустить выполнение макроса
- Нажмите "Stop Macro" чтобы остановить текущий макрос

### 5. Сохранение и загрузка пресетов
- Используйте "Save Preset" чтобы сохранить текущие настройки в файл JSON
- Используйте "Load Preset" чтобы загрузить ранее сохраненные настройки

## Режимы работы

### Режим мониторинга
1. Нажмите "Start Monitoring" чтобы начать отслеживание таймера игры
2. Текущее значение таймера будет отображаться в поле "Current Frame"
3. Программа автоматически начинает блокировку клавиш и выполнение макросов в зависимости от настроенных порогов

### Режим выполнения макросов
- При достижении таймером порога макроса (по умолчанию 280мс) начинается автоматическое выполнение последовательности нажатий
- Во время выполнения макроса физический ввод игрока игнорируется для предотвращения сбоя комбо

## Поддерживаемые устройства ввода
Программа поддерживает все типы устройств ввода через dinput8.dll, включая:
- Клавиатуры
- Геймпады
- Джойстики

## Частые проблемы и решения

**Проблема**: DLL не деплоится
**Решение**: Убедитесь, что вы выбрали правильный путь к MK10.exe

**Проблема**: Таймер не обновляется
**Решение**: Проверьте, что игра запущена и вы нажали "Start Monitoring"

**Проблема**: Макрос не выполняется
**Решение**: Проверьте, что вы добавили шаги в редактор макросов и таймер достиг нужного порога

## Безопасность
- Программа работает только локально
- Не отправляет данные в интернет
- Внедрение DLL происходит только в ваш процесс игры

## Языковая поддержка
Программа поддерживает русский и английский языки.
